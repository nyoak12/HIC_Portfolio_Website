Featured Project

Machine-to-Machine Learning Robotics

A robotics project integrating YOLOv8-based object detection with autonomous decision-making. The system enables real-time communication between multiple robotic agents using telemetry data and a central command module. Designed for search-and-rescue simulations, this project demonstrates applied machine learning in high-latency environments.

  • Built with Python, OpenCV, and Roboflow API
  • Integrated telemetry streaming using C++ DLL injection
  • Developed adaptive decision logic for multi-robot coordination
